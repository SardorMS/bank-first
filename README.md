# Bank-first
Function that returns the maximum payment, in Go (Golang)

# Installation

1. Clone this code into a directory:
 ```
 git clone https://github.com/SardorMS/bank-first
 ```

2. Create a module named for example - "bank-first":
 
 ```sh
 go mod init bank-first
 ``` 
 
3. To RUN Example test use:
```sh
go test -v ./...
```
